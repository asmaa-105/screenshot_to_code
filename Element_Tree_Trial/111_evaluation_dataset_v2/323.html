      トーク・音声通話・ビデオ通話

      ユーザー同士であれば、個人間はもちろん、複数人でも、トーク・音声通話・ビデオ通話を利用することができます。

    • スタンプ・絵文字・着せかえ

      スタンプ・絵文字・着せかえ

      トークがより楽しくなる多種多様なスタンプ・絵文字や、好みに合わせてLINEをカスタマイズできる着せかえがご利用できます。

    • オープンチャット

      オープンチャット

      LINEで繋がっていない興味関心事が近い人たちとも、日常生活に密着した話題についてトークを楽しむことができるサービスです。

    • ホームタブ

      ホームタブ

      友だちリストやLINEスタンプなど、LINEの様々な情報・サービス・コンテンツにアクセスできます。

    • LINE VOOM

      LINE VOOM

      スキマ時間に気軽に自分の動画を投稿したり、様々なジャンルのショート動画などを視聴して楽しむことができる動画プラットフォームです。

    • ウォレットタブ

      ウォレットタブ

      おトクに便利に使えるLINEウォレット。「LINE Pay」をはじめ「LINE証券」などの金融サービスや、「LINEポイント」、「LINEクーポン」などのおトク情報をご利用ください​。
